DESCRIPTION

v.ssn.sites This module finds the contributing catchment area for each point in a vector. Additionally the module will also calculate catchment statistics for the corresponding catchment of each point.

This module outputs:

  • a list of the available rasters

EXAMPLE

Some examples 
# Just a quick example
v.ssn.sites input=sites direction=direction
